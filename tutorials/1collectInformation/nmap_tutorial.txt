####################
        NMAP
####################

Nmap ("Network Mapper") is an open source tool for network exploration and security auditing. 

With nmap you can do:
- Server discovery
- Identify open ports
- Detect what services are running
- Detect what system operating have the machine

among other things.

Index

1 Ping scan
2 Ports scan


Ping scan
---------------------------

You can disovery all machines in your network with this scan, the next commands only print out the available hosts that responded at ping.

sudo nmap -sn 192.168.0.0/24

if you want more information you can add -v at all commands.


Ports scan
--------------------------

One time you have a list with the IPs you can list all ports to discovery the server in your network.

To scan all network you need execute:

sudo nmap -sS 192.168.0.0/24

To scan only a hosts do you want you can execute:

(With range)

sudo nmap -sS 192.168.0.0-20/24

(With only one target)

nmap -sS 192.168.0.13/24
